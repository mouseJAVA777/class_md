# 시프트 마이크로 연산

![image-20220630115001875](논리마이크로연산.assets/image-20220630115001875.png)

- 논리 시프트 : 오른쪽으로 시프트 하면 제일 오른쪽 비트는 사라지고 왼쪽에 0 비트가 들어감
- 순환 시프트 : 오른쪽으로 시프트 하면 제일 오른쪽 비트가 제일 왼쪽으로 들어감
- 산술 시프트 : 부호 비트를 제외하여 시프트함
